1. Desigualdad de Markov

Desigualdad de Markov: Sea \(X\) una variable aleatoria no negativa con media finita \(\mu_X\). Entonces, para todo \(c > 0\): \[ \mathbb{P}(X \ge c) \le \frac{\mathbb{E}[X]}{c} = \frac{\mu_X}{c} \]

Verificación interactiva

2.0
Demostración (caso continuo): \[ \mu_X = \int_0^\infty x f(x) dx \ge \int_c^\infty x f(x) dx \ge c \int_c^\infty f(x) dx = c \cdot \mathbb{P}(X \ge c) \] Por lo tanto, \(\mathbb{P}(X \ge c) \le \mu_X / c\).
Ejemplo: Si el tiempo medio de reparación es 6.2 horas, la probabilidad de que una reparación tome más de 22.5 horas es: \[ \mathbb{P}(X \ge 22.5) \le \frac{6.2}{22.5} \approx 0.2756 \]
¿Por qué Markov solo funciona para variables no negativas?
La demostración usa que \(x \ge c\) dentro de la integral \(\int_c^\infty\). Si \(X\) pudiera tomar valores negativos, la desigualdad no se cumpliría (por ejemplo, una normal con media 0 daría \(\mathbb{P}(X \ge c) > 0\) pero el lado derecho sería 0).

2. Desigualdad de Chebyshev

Desigualdad de Chebyshev: Sea \(X\) con media \(\mu\) y varianza \(\sigma^2\) finitas. Entonces, para todo \(\epsilon > 0\): \[ \mathbb{P}(|X - \mu| \ge \epsilon) \le \frac{\sigma^2}{\epsilon^2} \] Equivalentemente, para \(k > 0\): \[ \mathbb{P}(|X - \mu| \ge k\sigma) \le \frac{1}{k^2}, \quad \mathbb{P}(|X - \mu| < k\sigma) \ge 1 - \frac{1}{k^2} \]

Visualización: Cota de Chebyshev vs probabilidad real

2.0
Demostración:
Sea \(Y = (X - \mu)^2 \ge 0\). Aplicando Markov a \(Y\) con \(c = \epsilon^2\): \[ \mathbb{P}((X-\mu)^2 \ge \epsilon^2) \le \frac{\mathbb{E}[(X-\mu)^2]}{\epsilon^2} = \frac{\sigma^2}{\epsilon^2} \] Como \(\{(X-\mu)^2 \ge \epsilon^2\} = \{|X-\mu| \ge \epsilon\}\), se obtiene el resultado.
Ejemplo del PDF (tienda): Número de clientes por día: media 20, desviación 2. \[ \mathbb{P}(17 \le X \le 23) = \mathbb{P}(|X-20| < 3) \ge 1 - \frac{4}{9} \approx 0.5556 \] Con \(k=2\) (2 desviaciones): \(\mathbb{P}(|X-20| < 4) \ge 1 - 1/4 = 0.75\)
¿La cota de Chebyshev es siempre ajustada?
No, generalmente es conservadora. Pero puede alcanzarse, como en la distribución discreta del PDF con \(P(X=-1)=1/8, P(X=0)=6/8, P(X=1)=1/8\), donde \(\mathbb{P}(|X| \ge 1) = 1/4\) y la cota de Chebyshev es exactamente \(1/4\).

3. Chebyshev Unilateral (Cantelli)

La desigualdad de Chebyshev unilateral (o de Cantelli) acota la probabilidad de una cola: \[ \mathbb{P}(X - \mu \ge a) \le \frac{\sigma^2}{\sigma^2 + a^2}, \quad a > 0 \] Análogamente, \(\mathbb{P}(X - \mu \le -a) \le \frac{\sigma^2}{\sigma^2 + a^2}\).

Comparación: Chebyshev bilateral vs unilateral

0 1.0 2.0
Ejemplo del PDF (reparación de máquina):
Tiempo medio: \(\mu = 6.2\) h, desviación \(\sigma = 3.52\) h.
Empleado nuevo tarda 22.5 h → \(a = 16.3\). \[ \mathbb{P}(X \ge 22.5) = \mathbb{P}(X - \mu \ge 16.3) \le \frac{3.52^2}{3.52^2 + 16.3^2} = \frac{12.39}{12.39 + 265.69} = 0.0446 \] ¡Solo 4.46% de probabilidad! → El empleado probablemente necesita entrenamiento.
¿Por qué la cota unilateral es más ajustada que la bilateral?
La cota bilateral \(\mathbb{P}(|X-\mu| \ge a) \le \sigma^2/a^2\) considera ambas colas. Para una cola, la probabilidad es aproximadamente la mitad, y la cota unilateral \(\sigma^2/(\sigma^2+a^2)\) es más pequeña (mejor) que \(\sigma^2/a^2\) cuando \(a\) es grande.

4. Desigualdad de Jensen

Desigualdad de Jensen: Si \(h\) es una función convexa, entonces \[ \mathbb{E}[h(X)] \ge h(\mathbb{E}[X]) \] Si \(h\) es cóncava, la desigualdad se invierte: \[ \mathbb{E}[h(X)] \le h(\mathbb{E}[X]) \]

Verificación de Jensen para diferentes funciones

Demostración geométrica:
Para una función convexa \(h\), existe una recta soporte \(\ell(x) = a + bx\) que pasa por \((\mathbb{E}[X], h(\mathbb{E}[X]))\) y satisface \(h(x) \ge \ell(x)\) para todo \(x\).
Tomando esperanza: \[ \mathbb{E}[h(X)] \ge \mathbb{E}[\ell(X)] = a + b\mathbb{E}[X] = \ell(\mathbb{E}[X]) = h(\mathbb{E}[X]) \]
Ejemplo del PDF: \(X \sim \text{Unif}(0,1)\), \(h(x)=x^2\) (convexa). \[ \mathbb{E}[X] = \frac{1}{2}, \quad h(\mathbb{E}[X]) = \frac{1}{4} \] \[ \mathbb{E}[h(X)] = \mathbb{E}[X^2] = \frac{1}{3} \approx 0.3333 \ge 0.25 \] ¡Se cumple la desigualdad!
Aplicación importante: ¿Qué dice Jensen sobre \(\mathbb{E}[1/X]\) vs \(1/\mathbb{E}[X]\)?
La función \(h(x) = 1/x\) es convexa para \(x > 0\). Por Jensen: \[ \mathbb{E}[1/X] \ge 1/\mathbb{E}[X] \] Esto es útil en finanzas (el rendimiento esperado de un activo no es el recíproco del rendimiento esperado de su inverso).
