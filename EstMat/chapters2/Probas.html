1. Espacios de Probabilidad (Repaso)

Un espacio de probabilidad es una terna $(\Omega, \mathcal{F}, \mathbb{P})$ donde:
  • $\Omega$: espacio muestral (conjunto de resultados posibles)
  • $\mathcal{F}$: $\sigma$-álgebra (eventos, cerrada bajo complementos y uniones numerables)
  • $\mathbb{P}$: medida de probabilidad (K1: no negativa, K2: $\mathbb{P}(\Omega)=1$, K3: aditividad numerable)

Verificación de probabilidad condicional

Propiedades derivadas de los axiomas:
  • $\mathbb{P}(A^C) = 1 - \mathbb{P}(A)$
  • $\mathbb{P}(\emptyset) = 0$
  • $\mathbb{P}(A \cup B) = \mathbb{P}(A) + \mathbb{P}(B) - \mathbb{P}(A \cap B)$
  • Si $A \subseteq B$, entonces $\mathbb{P}(A) \le \mathbb{P}(B)$
Demostración del Teorema de Probabilidad Total:
Si $\{A_i\}$ es una partición de $\Omega$, entonces: \[ \mathbb{P}(B) = \sum_{i=1}^{\infty} \mathbb{P}(B|A_i)\mathbb{P}(A_i) \] Demostración: $B = B \cap \Omega = B \cap (\bigcup_i A_i) = \bigcup_i (B \cap A_i)$. Como los $A_i$ son ajenos, los $B \cap A_i$ también lo son. Por K3: \[ \mathbb{P}(B) = \sum_i \mathbb{P}(B \cap A_i) = \sum_i \mathbb{P}(B|A_i)\mathbb{P}(A_i) \]

2. Variables Aleatorias y Momentos

Variable aleatoria: Función $X: \Omega \to \mathbb{R}$ tal que $\{X \le x\} \in \mathcal{F}$ para todo $x \in \mathbb{R}$.
Valor esperado: $\mathbb{E}[X] = \int_{\Omega} X d\mathbb{P} = \begin{cases} \sum x_i f(x_i) & \text{discreto} \\ \int x f(x) dx & \text{continuo} \end{cases}$

Función Generadora de Momentos (FGM): \[ m_X(t) = \mathbb{E}[e^{tX}], \quad \text{si existe para } |t| < \delta \] Propiedad: $\mathbb{E}[X^r] = m_X^{(r)}(0)$
Función Característica (FC): $\varphi_X(t) = \mathbb{E}[e^{itX}]$ (siempre existe)
¿Cuándo no existe la FGM?
La FGM no existe si la integral $\int e^{tx}f(x)dx$ diverge para todo $t>0$. Ejemplos: Cauchy (no tiene momentos), Lognormal (tiene todos los momentos pero la FGM diverge). La función característica $\varphi_X(t)$ siempre existe.

3. Desigualdades Probabilísticas

Desigualdad de Markov: Para $X \ge 0$, $\mathbb{P}(X \ge k) \le \frac{\mathbb{E}[X]}{k}$
Desigualdad de Chebyshev: $\mathbb{P}(|X-\mu| \ge \epsilon) \le \frac{\sigma^2}{\epsilon^2}$
Desigualdad de Jensen: Si $h$ es convexa, $\mathbb{E}[h(X)] \ge h(\mathbb{E}[X])$

Verificación de Chebyshev

2.0
Demostración de Chebyshev a partir de Markov:
Sea $Y = (X - \mu)^2 \ge 0$. Por Markov con $c = \epsilon^2$: \[ \mathbb{P}((X-\mu)^2 \ge \epsilon^2) \le \frac{\mathbb{E}[(X-\mu)^2]}{\epsilon^2} = \frac{\sigma^2}{\epsilon^2} \] Como $(X-\mu)^2 \ge \epsilon^2 \iff |X-\mu| \ge \epsilon$, se obtiene el resultado.

4. Vectores Aleatorios y Covarianza

Para un vector aleatorio $\mathbf{X} = (X_1, \ldots, X_n)^T$: \[ \boldsymbol{\mu} = \mathbb{E}[\mathbf{X}], \quad \boldsymbol{\Sigma} = \operatorname{cov}(\mathbf{X}) = \mathbb{E}[(\mathbf{X}-\boldsymbol{\mu})(\mathbf{X}-\boldsymbol{\mu})^T] \] La covarianza entre $X_i$ y $X_j$ es $\sigma_{ij} = \mathbb{E}[X_i X_j] - \mathbb{E}[X_i]\mathbb{E}[X_j]$

Simulación de correlación

0.70
Esperanza y varianza condicional: \[ \mathbb{E}[Y|X=x] = \int y f(y|x) dy \] \[ \mathbb{E}[Y] = \mathbb{E}[\mathbb{E}[Y|X]], \quad \operatorname{var}(Y) = \operatorname{var}(\mathbb{E}[Y|X]) + \mathbb{E}[\operatorname{var}(Y|X)] \]
¿Qué significa $\operatorname{cov}(X,Y)=0$?
Significa que $X$ e $Y$ están no correlacionadas. Esto NO implica independencia (a menos que sean conjuntamente normales). El contraejemplo clásico: $X \sim N(0,1)$, $Y = X^2$ tiene covarianza cero pero son dependientes.

5. Transformaciones y Método Delta

Teorema de transformación (univariado): Si $Y = g(X)$ con $g$ monótona y $h = g^{-1}$, \[ f_Y(y) = f_X(h(y)) \left| \frac{dh}{dy} \right| \] Método Delta: Para $h$ dos veces diferenciable, \[ \mathbb{E}[h(X)] \approx h(\mu) + \frac{h''(\mu)}{2}\sigma^2, \quad \operatorname{var}(h(X)) \approx (h'(\mu))^2 \sigma^2 \]

Método Delta: Aproximación de $\mathbb{E}[\sqrt{X}]$

Expansión de Taylor (justificación del Método Delta): \[ h(X) = h(\mu) + h'(\mu)(X-\mu) + \frac{h''(\xi)}{2}(X-\mu)^2 \] Tomando esperanza: \[ \mathbb{E}[h(X)] = h(\mu) + \frac{h''(\mu)}{2}\sigma^2 + \text{términos de orden superior} \] \[ \operatorname{var}(h(X)) \approx (h'(\mu))^2 \sigma^2 \]
Ejemplo del PDF (Uniforme(0,1), $h(x)=\sqrt{x}$): \[ \mathbb{E}[\sqrt{X}] = \frac{2}{3} \approx 0.6667, \quad \text{Método Delta: } \sqrt{0.5} \approx 0.7071 \] \[ \operatorname{var}(\sqrt{X}) = 0.0556, \quad \text{Método Delta: } \left(\frac{1}{2\sqrt{0.5}}\right)^2 \cdot \frac{1}{12} = 0.0589 \]

¿Cuál es la distribución del mínimo de una muestra exponencial?
Si $X_i \sim \text{Exp}(\lambda)$, entonces $X_{(1)} \sim \text{Exp}(n\lambda)$. Esto tiene sentido porque la tasa de fallo del mínimo es la suma de las tasas individuales (mecanismo en serie: el sistema falla cuando falla el primer componente).
