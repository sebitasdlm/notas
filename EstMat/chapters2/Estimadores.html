1. Sesgo, Varianza y Error Cuadrático Medio (ECM)

Sesgo: $\operatorname{Bias}(\hat{\theta}) = \mathbb{E}[\hat{\theta}] - \theta$
Error Cuadrático Medio (ECM): $\operatorname{ECM}(\hat{\theta}) = \mathbb{E}[(\hat{\theta} - \theta)^2] = \operatorname{var}(\hat{\theta}) + \operatorname{Bias}^2(\hat{\theta})$
Un estimador es insesgado si $\operatorname{Bias}(\hat{\theta}) = 0$.

Simulación: Sesgo y varianza de estimadores

30
Demostración de la descomposición ECM: \[ \begin{align*} \operatorname{ECM}(\hat{\theta}) &= \mathbb{E}[(\hat{\theta} - \theta)^2] \\ &= \mathbb{E}[((\hat{\theta} - \mathbb{E}[\hat{\theta}]) + (\mathbb{E}[\hat{\theta}] - \theta))^2] \\ &= \operatorname{var}(\hat{\theta}) + (\mathbb{E}[\hat{\theta}] - \theta)^2 + 2(\mathbb{E}[\hat{\theta}] - \theta)\mathbb{E}[\hat{\theta} - \mathbb{E}[\hat{\theta}]] \\ &= \operatorname{var}(\hat{\theta}) + \operatorname{Bias}^2(\hat{\theta}) \end{align*} \]
¿Puede un estimador sesgado tener menor ECM que uno insesgado?
¡Sí! Es el trade-off sesgo-varianza. Un estimador puede tener un pequeño sesgo pero una varianza mucho menor, resultando en un ECM menor. Esto es común en métodos de regularización como Ridge regression.

2. Método de Momentos (EMM)

El método de momentos iguala momentos poblacionales con momentos muestrales. \[ \mu_r = \mathbb{E}[X^r], \quad m_r = \frac{1}{n}\sum_{i=1}^n X_i^r \] Se resuelve $\mu_r(\theta) = m_r$ para $r=1,\ldots,k$ para obtener $\tilde{\theta}_{\text{EMM}}$.

Estimación por Momentos: Gamma

2.0 1.0 100
Para Gamma(α,β): \[ \mu_1 = \alpha\beta, \quad \mu_2 = \alpha\beta^2 + \alpha^2\beta^2 \] Resolviendo: $\tilde{\alpha}_{\text{EMM}} = \frac{m_1^2}{m_2 - m_1^2}$, $\tilde{\beta}_{\text{EMM}} = \frac{m_2 - m_1^2}{m_1}$

3. Método de Máxima Verosimilitud (EMV)

La función de verosimilitud es $L(\theta; \mathbf{x}) = \prod_{i=1}^n f(x_i; \theta)$.
El estimador de máxima verosimilitud (EMV) es $\hat{\theta}_{\text{EMV}} = \arg\max_{\theta} L(\theta; \mathbf{x})$.
Equivalentemente, maximiza $\ell(\theta; \mathbf{x}) = \log L(\theta; \mathbf{x})$.

EMV para diferentes distribuciones

50
EMV para Poisson(λ): $\hat{\lambda}_{\text{EMV}} = \bar{X}_n$
EMV para Exponencial(θ): $\hat{\theta}_{\text{EMV}} = \bar{X}_n$
EMV para Uniforme(0,θ): $\hat{\theta}_{\text{EMV}} = X_{(n)}$ (el máximo muestral)
EMV para Normal(μ,σ²): $\hat{\mu}_{\text{EMV}} = \bar{X}_n$, $\hat{\sigma}^2_{\text{EMV}} = \frac{1}{n}\sum (X_i - \bar{X}_n)^2$
¿El EMV de σ² en Normal es insesgado?
No. $\mathbb{E}[\hat{\sigma}^2_{\text{EMV}}] = \frac{n-1}{n}\sigma^2$, tiene sesgo $-\sigma^2/n$. El estimador insesgado es $S^2 = \frac{n}{n-1}\hat{\sigma}^2_{\text{EMV}}$. Sin embargo, $\hat{\sigma}^2_{\text{EMV}}$ es consistente (el sesgo tiende a 0 cuando $n\to\infty$).

4. Información de Fisher y Cota de Cramér-Rao

Información de Fisher: \[ I(\theta) = \mathbb{E}\left[\left(\frac{\partial}{\partial\theta}\log f(X;\theta)\right)^2\right] = -\mathbb{E}\left[\frac{\partial^2}{\partial\theta^2}\log f(X;\theta)\right] \] Cota de Cramér-Rao: Para cualquier estimador insesgado $\hat{\theta}$, \[ \operatorname{var}(\hat{\theta}) \ge \frac{1}{n I(\theta)} \]

Verificación: EMV alcanza la cota

50
Información de Fisher para Exponencial(λ): \[ \log f(x;\lambda) = \log\lambda - \lambda x, \quad \frac{\partial}{\partial\lambda}\log f = \frac{1}{\lambda} - x \] \[ I(\lambda) = \mathbb{E}\left[\left(\frac{1}{\lambda} - X\right)^2\right] = \operatorname{var}(X) = \frac{1}{\lambda^2} \] Cota: $\operatorname{var}(\hat{\lambda}) \ge \frac{1}{n I(\lambda)} = \frac{\lambda^2}{n}$

5. Consistencia y Normalidad Asintótica del EMV

Bajo condiciones de regularidad, el EMV es consistente y asintóticamente normal: \[ \sqrt{n}(\hat{\theta}_n - \theta_0) \xrightarrow{D} N\left(0, \frac{1}{I(\theta_0)}\right) \] Es decir, para $n$ grande, $\hat{\theta}_n \approx N\left(\theta_0, \frac{1}{n I(\theta_0)}\right)$.

Verificación: Normalidad asintótica del EMV

50
Interpretación: La varianza asintótica del EMV es $\frac{1}{nI(\theta_0)}$, que es la cota de Cramér-Rao. El EMV es asintóticamente eficiente: alcanza la mínima varianza posible para muestras grandes.

6. Principio de Invarianza del EMV

Si $\hat{\theta}$ es el EMV de $\theta$, entonces para cualquier función $\tau$, el EMV de $\tau(\theta)$ es $\tau(\hat{\theta})$.
Esto se debe a que la función de verosimilitud se maximiza en el mismo punto.

Ejemplo: EMV de la varianza en Bernoulli

0.30 50
Para Bernoulli(p), $\operatorname{var}(X) = p(1-p)$. Por invarianza, \[ \widehat{\operatorname{var}(X)} = \hat{p}(1-\hat{p}) = \bar{X}_n(1-\bar{X}_n) \] Este estimador es sesgado (porque $\mathbb{E}[\bar{X}_n^2] = p^2 + p(1-p)/n$), pero consistente.
¿La invarianza se cumple para cualquier función τ?
Sí, para cualquier función medible. Sin embargo, si τ no es uno-a-uno, puede haber múltiples EMV. Por ejemplo, para $X \sim N(\mu,1)$, $\hat{\mu}_{\text{EMV}} = \bar{X}$, entonces el EMV de $\mu^2$ es $\bar{X}^2$, aunque $\mu^2$ no sea uno-a-uno.
