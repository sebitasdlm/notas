📊 Ejemplo: WebR - R en el navegador

Este código R se ejecuta directamente en tu navegador usando WebR.

Código R: 
# Simular la Ley de Grandes Números
set.seed(123)
medias <- sapply(1:500, function(n) mean(rnorm(n, mean=10, sd=2)))
plot(1:500, medias, type="l", xlab="n", ylab="Media muestral", main="Convergencia a μ=10")
abline(h=10, col="red", lwd=2)

# Calcular algunas estadísticas
x <- rnorm(1000, mean=50, sd=10)
cat("Media:", mean(x), "\n")
cat("Desviación estándar:", sd(x), "\n")
cat("Percentil 95%:", quantile(x, 0.95), "\n")
