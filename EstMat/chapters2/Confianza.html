1. IC para la Media (Varianza Conocida)

Intervalo de confianza (1-α)% para μ: \[ \bar{X} \pm z_{1-\alpha/2} \frac{\sigma}{\sqrt{n}} \] Cantidad pivotal: $Z = \frac{\bar{X} - \mu}{\sigma/\sqrt{n}} \sim N(0,1)$

Simulación: Cobertura del IC

0 1.0 30
Interpretación frecuentista:
Si repetimos el experimento muchas veces, aproximadamente 100(1-α)% de los intervalos construidos contendrán el verdadero valor de μ.
Para una muestra observada, el intervalo es fijo; la confianza se refiere al proceso.

2. IC para la Media (Varianza Desconocida) - t-Student

Cantidad pivotal: \[ T = \frac{\bar{X} - \mu}{S/\sqrt{n}} \sim t_{n-1} \] IC (1-α)% para μ: \[ \bar{X} \pm t_{1-\alpha/2, n-1} \frac{S}{\sqrt{n}} \]

Simulación: IC con t-Student vs Normal

0 1.0 10
Comparación de anchos:
Para α=0.05, n=10: $t_{0.975,9} = 2.262$, $z_{0.975} = 1.96$
El intervalo t es más ancho porque incorpora la incertidumbre de estimar σ. Para n grande, $t_{n-1} \approx z$.

3. IC para la Varianza (Normal)

Cantidad pivotal: \[ Q = \frac{(n-1)S^2}{\sigma^2} \sim \chi^2_{n-1} \] IC (1-α)% para σ²: \[ \left( \frac{(n-1)S^2}{\chi^2_{1-\alpha/2, n-1}}, \frac{(n-1)S^2}{\chi^2_{\alpha/2, n-1}} \right) \]

Simulación: IC para la varianza

1.0 30
Nota: El IC para σ² no es simétrico porque la distribución χ² no es simétrica. La elección de cuantiles con colas iguales (α/2) es convencional pero no óptima en términos de longitud mínima.

4. Comparación de Poblaciones Normales

IC para diferencia de medias (varianzas iguales): \[ S_p^2 = \frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1+n_2-2} \] \[ (\bar{X}-\bar{Y}) \pm t_{1-\alpha/2, n_1+n_2-2} \cdot S_p \sqrt{\frac{1}{n_1} + \frac{1}{n_2}} \] IC para razón de varianzas (F): \[ \left( \frac{S_1^2/S_2^2}{F_{1-\alpha/2, n_1-1, n_2-1}}, \frac{S_1^2/S_2^2}{F_{\alpha/2, n_1-1, n_2-1}} \right) \]

Simulación: Comparación de medias

0 1 1.0 20 20
¿Qué significa que el IC contenga al 0?
Si el intervalo de confianza para μ₁-μ₂ contiene al 0, no podemos rechazar la hipótesis nula de que las medias son iguales con el nivel de confianza correspondiente. Es equivalente a una prueba de hipótesis bilateral.

5. IC para Proporciones (Teorema Central del Límite)

IC aproximado (1-α)% para p (Bernoulli): \[ \hat{p} \pm z_{1-\alpha/2} \sqrt{\frac{\hat{p}(1-\hat{p})}{n}} \] IC para diferencia de proporciones: \[ (\hat{p}_1 - \hat{p}_2) \pm z_{1-\alpha/2} \sqrt{\frac{\hat{p}_1(1-\hat{p}_1)}{n_1} + \frac{\hat{p}_2(1-\hat{p}_2)}{n_2}} \]

Condiciones de validez: Para que la aproximación normal sea razonable, se recomienda $n\hat{p} \ge 5$ y $n(1-\hat{p}) \ge 5$.
En caso contrario, usar métodos exactos (Clopper-Pearson) o corrección de continuidad.

6. IC Basados en EMV (Aproximación Normal)

Para el EMV $\hat{\theta}$, bajo condiciones de regularidad: \[ \hat{\theta} \approx N\left(\theta, \frac{1}{nI(\theta)}\right) \] IC aproximado (1-α)%: \[ \hat{\theta} \pm z_{1-\alpha/2} \cdot \frac{1}{\sqrt{nI(\hat{\theta})}} \]

Ejemplo: IC para λ en Poisson

4.0 50
Para Poisson(λ): \[ I(\lambda) = \frac{1}{\lambda}, \quad \text{IC: } \bar{X} \pm z_{1-\alpha/2} \sqrt{\frac{\bar{X}}{n}} \] Este intervalo es equivalente al IC para proporciones (p=λ cuando la varianza es λ).
