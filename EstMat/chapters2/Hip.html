1. Errores Tipo I y II, Potencia de una Prueba

Error Tipo I: Rechazar $H_0$ cuando es verdadera. $\alpha = \mathbb{P}(\text{Rechazar } H_0 | H_0)$
Error Tipo II: No rechazar $H_0$ cuando es falsa. $\beta = \mathbb{P}(\text{No rechazar } H_0 | H_1)$
Potencia: $1 - \beta = \mathbb{P}(\text{Rechazar } H_0 | H_1)$
Nivel de significancia: $\alpha$ (probabilidad de error tipo I)

Simulador: Prueba de media con $\sigma$ conocida

Relación α-β: Para un tamaño de muestra fijo, disminuir α aumenta β y viceversa. Aumentar n reduce ambos errores simultáneamente.
¿Qué significa un valor-p pequeño?
El valor-p es la probabilidad de obtener un resultado tan extremo o más que el observado, suponiendo que $H_0$ es verdadera. Un valor-p pequeño (típicamente < 0.05) indica evidencia en contra de $H_0$.

2. Lema de Neyman-Pearson (Prueba Más Potente)

Para contrastar $H_0: \theta = \theta_0$ vs $H_1: \theta = \theta_1$ (hipótesis simples), la prueba más potente de nivel $\alpha$ tiene región de rechazo: \[ \mathcal{R} = \left\{ \mathbf{x} : \frac{L(\theta_0; \mathbf{x})}{L(\theta_1; \mathbf{x})} \le c \right\} \] donde $c$ se elige para que $\mathbb{P}_0(\mathcal{R}) = \alpha$.

Ejemplo: Prueba de dos monedas

0.50 0.70 10 7
Interpretación: El cociente de verosimilitudes $\Lambda = L(\theta_0)/L(\theta_1)$. Si $\Lambda < 1$, los datos son más probables bajo $H_1$; si $\Lambda > 1$, bajo $H_0$.

3. Pruebas Uniformemente Más Potentes (PUMP)

Una prueba es uniformemente más potente (PUMP) de nivel $\alpha$ si es la más potente para todo $\theta \in \Theta_1$. Para familias con razón de verosimilitud monótona (MLR), existen PUMP para hipótesis unilaterales.

Ejemplo: Normal con $\sigma$ conocida

0.0 1.0 25
Para $X \sim N(\mu, \sigma^2)$ con $\sigma$ conocida, la PUMP para $H_0: \mu \le \mu_0$ vs $H_1: \mu > \mu_0$ es: \[ \text{Rechazar } H_0 \text{ si } \bar{X} > \mu_0 + z_{1-\alpha} \frac{\sigma}{\sqrt{n}} \]

4. Relación entre Intervalos de Confianza y Pruebas de Hipótesis

Existe una dualidad entre intervalos de confianza y pruebas de hipótesis: \[ \text{Un IC de nivel } 1-\alpha \text{ para } \theta \text{ contiene a } \theta_0 \iff \text{no rechazamos } H_0: \theta = \theta_0 \text{ con nivel } \alpha \]

Verificación: IC vs Prueba para la media

IC para $\mu$ (con $\sigma$ conocida): $\bar{X} \pm z_{1-\alpha/2} \frac{\sigma}{\sqrt{n}}$
Prueba para $H_0: \mu = \mu_0$: rechazar si $|\bar{X} - \mu_0| > z_{1-\alpha/2} \frac{\sigma}{\sqrt{n}}$
¡Son exactamente equivalentes!

5. Cociente de Verosimilitud Generalizado (CVG)

Para hipótesis compuestas, el CVG se define como: \[ \Lambda(\mathbf{X}) = \frac{\sup_{\theta \in \Theta_0} L(\theta; \mathbf{X})}{\sup_{\theta \in \Theta} L(\theta; \mathbf{X})} \] Bajo $H_0$ y condiciones de regularidad, $-2\log\Lambda(\mathbf{X}) \xrightarrow{D} \chi^2_{\text{df}}$, donde df = dim($\Theta$) - dim($\Theta_0$).

Ejemplo: CVG para media normal (varianza desconocida)

0.0 0.5 1.0 50
Caso Normal (varianza desconocida): \[ -2\log\Lambda = n\log\left(1 + \frac{n(\bar{X} - \mu_0)^2}{(n-1)S^2}\right) \approx \frac{(\bar{X} - \mu_0)^2}{S^2/n} \sim \chi^2_1 \] Esta es la base de la prueba t de Student (aunque se usa t-student exacta en lugar de χ²).

6. Prueba χ² de Bondad de Ajuste

Para probar si los datos siguen una distribución específica: \[ X^2 = \sum_{i=1}^k \frac{(O_i - E_i)^2}{E_i} \xrightarrow{D} \chi^2_{k-1-m} \] donde $m$ es el número de parámetros estimados.
Regla: Rechazar $H_0$ si $X^2 > \chi^2_{1-\alpha, k-1-m}$.

¿Cuándo es válida la aproximación χ²?
Se recomienda que todos los valores esperados $E_i$ sean al menos 5. Si no, se deben agrupar categorías adyacentes.

7. Gráficas de Probabilidad (Q-Q Plots)

Las gráficas de probabilidad (Q-Q plots) son una herramienta visual para evaluar si una muestra proviene de una determinada distribución.
Idea: Graficar los cuantiles muestrales $X_{(j)}$ vs los cuantiles teóricos $F^{-1}\left(\frac{j}{n+1}\right)$. Si los puntos se alinean aproximadamente en una recta, el ajuste es razonable.

50
Teorema de transformación integral: Si $X$ tiene distribución continua $F$, entonces $F(X) \sim Unif(0,1)$.
Por lo tanto, $X_{(j)} \approx F^{-1}\left(\frac{j}{n+1}\right)$.
¿Qué indica una desviación sistemática de la recta en un Q-Q plot?
Una curvatura sugiere asimetría; una forma de "S" sugiere colas más pesadas o más livianas que la distribución teórica; una desviación en los extremos indica outliers.

8. Función de Distribución Empírica (FDE)

La función de distribución empírica (FDE) se define como: \[ F_n(x) = \frac{1}{n} \sum_{i=1}^n \mathbf{1}_{(-\infty, x]}(X_i) = \frac{\#\{X_i \le x\}}{n} \] Propiedades:
  • $\mathbb{E}[F_n(x)] = F(x)$ (insesgado)
  • $\operatorname{var}(F_n(x)) = \frac{1}{n}F(x)(1-F(x))$
  • $\sqrt{n}(F_n(x)-F(x)) \xrightarrow{D} N(0, F(x)(1-F(x)))$

Visualización: FDE vs Distribución Teórica

100
Teorema de Glivenko-Cantelli: \[ \mathbb{P}\left( \sup_{x \in \mathbb{R}} |F_n(x) - F(x)| \to 0 \right) = 1 \] La convergencia de $F_n$ a $F$ es uniforme y casi segura.

9. Prueba de Kolmogorov-Smirnov (KS)

El estadístico de Kolmogorov-Smirnov mide la distancia máxima entre $F_n$ y $F_0$: \[ D_n = \sup_{x \in \mathbb{R}} |F_n(x) - F_0(x)| \] Bajo $H_0: F = F_0$, la distribución de $\sqrt{n}D_n$ es conocida (libre de distribución).
Regla: Rechazar $H_0$ si $D_n > d_{\alpha, n}$ (valor crítico tabulado).

Valores críticos aproximados para KS ($\alpha=0.05$): \[ d_{0.05, n} \approx \frac{1.36}{\sqrt{n}}, \quad d_{0.01, n} \approx \frac{1.63}{\sqrt{n}} \] Para $n=50$, $d_{0.05} \approx 0.192$.
¿Qué ventaja tiene KS sobre la prueba χ² de bondad de ajuste?
KS es no paramétrica (libre de distribución) y no requiere agrupar datos en categorías. Sin embargo, no es aplicable cuando los parámetros se estiman de los datos (se requiere la prueba de Lilliefors). También es más sensible a diferencias en el centro de la distribución.

10. Comparación de Pruebas de Bondad de Ajuste

Resumen de pruebas para bondad de ajuste:
  • χ² de Pearson: Para datos agrupados, requiere $E_i \ge 5$.
  • Kolmogorov-Smirnov: Para datos continuos, basado en $D_n = \sup|F_n-F_0|$.
  • Anderson-Darling: Da más peso a las colas.
  • Shapiro-Wilk: Específica para normalidad.

Recomendaciones prácticas:
  • Para muestras pequeñas (n < 30), usar KS (distribución exacta).
  • Para muestras grandes, χ² es más potente cuando los datos son discretos.
  • Si se estiman parámetros (ej. Normal con µ,σ desconocidos), usar Lilliefors.
