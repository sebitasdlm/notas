Instrucciones:
Los ejemplos incluidos en las carpetas de univariado y multivariado sirven de guía. Para crear un nuevo curso basta clonar una de estas estructuras y editar los archivos:

index.html
Editar sólo el nombre del curso en los tags title y header.

cover.html
Esta es la página de presentación de las notas del curso. Todo el texto dentro del tag main es editable.

capítulos
Edita los capítulos en los que deseas contribuir directamente.

Si es necesario crear un capítulo nuevo:

Crear el archivo dentro de la carpeta chapters del curso correspondiente con un nombre simple.
Actualiza el archivo chapters.json para que el capítulo sea incluido en el índice del curso, aquí puedes incluir el título por extenso.
