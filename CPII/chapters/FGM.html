1. Función Generadora de Momentos

La función generadora de momentos (FGM) de una variable aleatoria \(X\) es \[ m_X(t) = \mathbb{E}[e^{tX}] \] siempre que la esperanza exista para \(|t| < \delta\) con \(\delta > 0\). Su nombre viene de que genera los momentos: \[ \mathbb{E}[X^r] = \left. \frac{d^r m_X(t)}{dt^r} \right|_{t=0} \]

Explora las FGM de diferentes distribuciones

Propiedades:
  • \(m_X(0) = 1\)
  • Si \(Y = a + bX\), entonces \(m_Y(t) = e^{at} m_X(bt)\)
  • Si dos variables tienen la misma FGM en un entorno de 0, tienen la misma distribución (Teorema de unicidad)
¿Pueden dos distribuciones diferentes tener los mismos momentos?
¡Sí! La distribución lognormal y una versión "perturbada" tienen exactamente los mismos momentos para todo r = 1,2,... pero son distribuciones distintas. Por eso la FGM es más poderosa: cuando existe, sí determina la distribución de manera única.

2. Suma de Variables Independientes

Si \(X\) y \(Y\) son independientes, entonces \[ m_{X+Y}(t) = m_X(t) \cdot m_Y(t) \] Esto hace que las FGM sean una herramienta poderosa para encontrar la distribución de sumas.

Simulador: Suma de dos variables independientes

Ejemplos clásicos (por unicidad):
  • \(X_i \sim \text{Ber}(p)\) independientes ⇒ \(\sum X_i \sim \text{Bin}(n,p)\)
  • \(X_i \sim \text{Po}(\lambda_i)\) independientes ⇒ \(\sum X_i \sim \text{Po}(\sum \lambda_i)\)
  • \(X_i \sim \text{N}(\mu_i, \sigma_i^2)\) independientes ⇒ \(\sum X_i \sim \text{N}(\sum \mu_i, \sum \sigma_i^2)\)
  • \(X_i \sim \text{Gamma}(\alpha_i, \beta)\) independientes ⇒ \(\sum X_i \sim \text{Gamma}(\sum \alpha_i, \beta)\)
¿Qué distribución tiene la suma de dos normales independientes?
¡Normal! Si \(X \sim N(\mu_X, \sigma_X^2)\) e \(Y \sim N(\mu_Y, \sigma_Y^2)\) independientes, entonces \(X+Y \sim N(\mu_X + \mu_Y, \sigma_X^2 + \sigma_Y^2)\). Esto se demuestra fácilmente porque \(m_{X+Y}(t) = e^{\mu_X t + \frac{1}{2}\sigma_X^2 t^2} \cdot e^{\mu_Y t + \frac{1}{2}\sigma_Y^2 t^2} = e^{(\mu_X+\mu_Y)t + \frac{1}{2}(\sigma_X^2+\sigma_Y^2)t^2}\).

3. FGM Conjunta (Caso Multivariado)

Para un vector aleatorio \((X,Y)\), la FGM conjunta es \[ m_{XY}(u,v) = \mathbb{E}[e^{uX + vY}] \] A partir de ella podemos obtener:
  • FGM marginales: \(m_X(u) = m_{XY}(u,0)\), \(m_Y(v) = m_{XY}(0,v)\)
  • Momentos cruzados: \(\mathbb{E}[X^r Y^s] = \left. \frac{\partial^{r+s}}{\partial u^r \partial v^s} m_{XY}(u,v) \right|_{u=0,v=0}\)
  • Independencia: \(X \perp Y \iff m_{XY}(u,v) = m_X(u) m_Y(v)\)

¿Cómo se detecta independencia con la FGM conjunta?
Si \(m_{XY}(u,v) = m_X(u) \cdot m_Y(v)\) para todo \(u,v\) en un entorno de (0,0), entonces \(X\) y \(Y\) son independientes. Esto es una consecuencia directa del teorema de unicidad.

4. Función Característica

La función característica (FC) se define como \[ \varphi_X(t) = \mathbb{E}[e^{itX}] = \mathbb{E}[\cos(tX)] + i\mathbb{E}[\sin(tX)] \] A diferencia de la FGM, siempre existe para todo \(t \in \mathbb{R}\). También determina la distribución de manera única.

Compara FGM y FC para diferentes distribuciones

Ejemplos de funciones características:
  • \(X \sim N(\mu,\sigma^2)\) ⇒ \(\varphi_X(t) = e^{i\mu t - \frac{1}{2}\sigma^2 t^2}\)
  • \(X \sim \text{Po}(\lambda)\) ⇒ \(\varphi_X(t) = e^{\lambda(e^{it}-1)}\)
  • \(X \sim \text{Bin}(n,p)\) ⇒ \(\varphi_X(t) = (q + pe^{it})^n\)
  • \(X \sim \text{Cauchy}(0,1)\) ⇒ \(\varphi_X(t) = e^{-|t|}\) (¡no tiene FGM!)
La distribución Cauchy tiene FC real. ¿Qué significa esto?
Una FC es real si y solo si la distribución es simétrica alrededor de 0. La Cauchy(0,1) es simétrica, por eso \(\varphi(t) = e^{-|t|}\) es real. Además, ¡la Cauchy no tiene FGM porque su esperanza no existe!
Relación FGM ↔ FC: Si la FGM existe, entonces \(\varphi_X(t) = m_X(it)\). Pero la FC puede existir aunque la FGM no (como en la distribución Cauchy).
