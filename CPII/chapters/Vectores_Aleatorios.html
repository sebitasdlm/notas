1. Espacios de Probabilidad

La base de todo: Un espacio de probabilidad es el escenario donde ocurre lo aleatorio. Tiene tres elementos: un espacio muestral \(\Omega\) (todos los resultados posibles), una colección de eventos \(\mathcal{F}\) (subconjuntos de \(\Omega\) que tienen probabilidad), y una medida de probabilidad \(\mathbb{P}\) que asigna números entre 0 y 1 a los eventos.

Visualiza un experimento simple: lanzar una moneda

🪙
\(\Omega = \{\text{Águila}, \text{Sol}\}\), \(\mathcal{F} = \{\emptyset, \{\text{A}\}, \{\text{S}\}, \Omega\}\), \(\mathbb{P}(\{\text{A}\}) = \mathbb{P}(\{\text{S}\}) = 0.5\) (moneda justa).
Axiomas de Kolmogorov (K1-K3):
K1: \(\mathbb{P}(A) \ge 0\) para todo evento \(A\).
K2: \(\mathbb{P}(\Omega) = 1\).
K3: Si \(A_1, A_2, \dots\) son eventos ajenos (\(A_i \cap A_j = \emptyset\)), entonces \(\mathbb{P}(\bigcup_{i=1}^{\infty} A_i) = \sum_{i=1}^{\infty} \mathbb{P}(A_i)\).
Consecuencia: De los axiomas se deduce que \(\mathbb{P}(A^c) = 1 - \mathbb{P}(A)\). ¿Por qué?
Como \(A\) y \(A^c\) son ajenos y \(A \cup A^c = \Omega\), por K2 y K3: \(1 = \mathbb{P}(\Omega) = \mathbb{P}(A) + \mathbb{P}(A^c)\), despejando se obtiene la fórmula.

2. Vectores Aleatorios Discretos

Un vector aleatorio \((X,Y)\) es una función que asigna un par ordenado de números a cada resultado del experimento. En el caso discreto, solo puede tomar un número finito o numerable de pares.

Propiedades: \(f_{XY}(x,y) \ge 0\) y \(\sum_x \sum_y f_{XY}(x,y) = 1\).

Pregunta: Al aumentar la carga, cambia \(\mathbb{P}(X=3)\)? ¿Cambia \(\mathbb{P}(X=Y)\)?
\(\mathbb{P}(X=3)\) no cambia (sigue siendo 1/6). Las marginales son invariantes.
\(\mathbb{P}(X=Y)\) sí aumenta con la carga.

\(f_X(x) = \sum_y f_{XY}(x,y)\)    y    \(f_Y(y) = \sum_x f_{XY}(x,y)\).

3. Función de Probabilidad Acumulada Conjunta

La función de distribución acumulada (FPA) se obtiene acumulando toda la probabilidad hacia abajo y hacia la izquierda desde el punto (x,y): \(F_{XY}(x,y) = \mathbb{P}(X \le x, Y \le y)\).

Visualiza F_{XY} para dados

x = 3 y = 3
Propiedades clave de \(F_{XY}\):
1. No decreciente en cada argumento.
2. \(F(-\infty, y)=0\), \(F(x, -\infty)=0\), \(F(\infty, \infty)=1\).
3. \(F_X(x) = \lim_{y\to\infty} F_{XY}(x,y)\).
4. Para \(a < b\), \(c < d\): \(\mathbb{P}(a < X \le b, c < Y \le d) = F(b,d) - F(b,c) - F(a,d) + F(a,c)\).

¿Toda función que cumple las propiedades es una FPA válida?

La prueba de fuego: Para una FPA válida, debe cumplirse que \(F(b,d)-F(b,c)-F(a,d)+F(a,c) \ge 0\) para todo \(a<b, c<d\).
Toma \(a=-1, b=3, c=-1, d=3\). En el segundo ejemplo: \(F(3,3)-F(3,-1)-F(-1,3)+F(-1,-1) = 1 - 1 - 1 + 0 = -1\), que no es ≥ 0. ¡No puede ser una FPA!

4. Vectores Aleatorios Continuos

Un vector continuo puede tomar cualquier valor en una región del plano. Su comportamiento se describe mediante una función de densidad conjunta \(f_{XY}(x,y) \ge 0\) tal que \(\iint_{\mathbb{R}^2} f_{XY}(x,y)\,dx\,dy = 1\).

Ejemplo 1: Distribución uniforme en el cuadrado [0,1]×[0,1]

Densidad: \(f_{XY}(x,y) = 1\) si \(0 \le x \le 1, 0 \le y \le 1\).
Marginales: \(f_X(x)=1\) en [0,1] (uniforme), igual para \(Y\).
\(F_{XY}(x,y) = xy\) en el cuadrado.

Ejemplo 2: Densidad con soporte triangular (de los apuntes)

\(f(x,y) = 2e^{-(x+y)}\) para \(0 \le x \le y < \infty\).

1.00
Marginales de este ejemplo:
\(f_X(x) = 2e^{-2x}\) (exponencial con media 1/2)
\(f_Y(y) = 2e^{-y}(1 - e^{-y})\).
Observa: El soporte es \(x \le y\). ¿Qué forma tiene? ¿Es un rectángulo?
No es un rectángulo, es un triángulo (infinito). Esto ya nos anticipa que X y Y no son independientes.

5. Independencia de Variables Aleatorias

Dos variables \(X\) y \(Y\) son independientes si conocer el valor de una no da información sobre la otra. Equivalentemente: \(F_{XY}(x,y) = F_X(x)F_Y(y)\) para todo \(x,y\).

Lema de factorización (para variables continuas)

\(X\) y \(Y\) son independientes si y solo si \(f_{XY}(x,y) = g(x)h(y)\) para algunas funciones \(g\) y \(h\).

Regla práctica: Si el soporte (región donde \(f>0\)) no es un producto cartesiano (rectángulo), entonces es imposible que sean independientes. Si es un rectángulo, aún hay que verificar la factorización.
Desafío: En el módulo 2 (tabla de dados), ¿cuándo son X e Y independientes?
Son independientes cuando la probabilidad conjunta es producto de las marginales. En el caso de dados honestos (\(f_{XY}=1/36\)) sí lo son. En los dados cargados (con carga >1) no son independientes.
Propiedad importante: Si \(X\) y \(Y\) son independientes, entonces para cualquier par de funciones \(g\) y \(h\), las variables \(g(X)\) y \(h(Y)\) también son independientes.
