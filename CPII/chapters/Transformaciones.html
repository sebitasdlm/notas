1. Transformación de Variables Aleatorias (Caso Univariado)

Si \(X\) es una variable aleatoria continua con densidad \(f_X\) y \(Y = g(X)\) es una transformación monótona (creciente o decreciente) y derivable, entonces: \[ f_Y(y) = f_X(h(y)) \left| \frac{dh(y)}{dy} \right|, \quad \text{donde } h = g^{-1} \] Si \(g\) no es monótona, se particiona el soporte en regiones donde sí lo sea: \[ f_Y(y) = \sum_{k=1}^{m} f_X(h_k(y)) \left| \frac{dh_k(y)}{dy} \right| \]

Simulador: Transformación de una uniforme a exponencial

1.0
Derivación: Transformación Uniforme → Exponencial
Sea \(U \sim \text{Unif}(0,1)\) y definimos \(Y = -\frac{1}{\lambda} \log(1-U)\).
\(F_U(u) = u\) para \(0 \(\left| \frac{dh}{dy} \right| = \lambda e^{-\lambda y}\).
Entonces \(f_Y(y) = f_U(h(y)) \cdot \lambda e^{-\lambda y} = 1 \cdot \lambda e^{-\lambda y}\).
¡Es la densidad de una exponencial con tasa λ!
Ejemplo del PDF: Transformación de una normal a lognormal
Si \(Y \sim N(\mu, \sigma^2)\) y \(X = e^Y\), entonces: \[ f_X(x) = \frac{1}{x\sqrt{2\pi}\sigma} \exp\left\{-\frac{1}{2}\left(\frac{\log x - \mu}{\sigma}\right)^2\right\}, \quad x > 0 \] Esta es la distribución Lognormal.
¿Qué pasa si aplicamos la transformación \(Y = X^2\) a una normal estándar?
Si \(Z \sim N(0,1)\) y \(Y = Z^2\), la transformación no es monótona (tiene dos ramas: \(z = \pm\sqrt{y}\)). \[ f_Y(y) = \frac{1}{2\sqrt{y}} [f_Z(\sqrt{y}) + f_Z(-\sqrt{y})] = \frac{1}{\sqrt{2\pi y}} e^{-y/2}, \quad y>0 \] ¡Esta es la densidad de una \(\chi^2_1\) (Gamma con α=1/2, β=2)!

2. Distribución Lognormal

Una variable \(X\) tiene distribución lognormal con parámetros \(\mu\) y \(\sigma^2\) si \(\log X \sim N(\mu, \sigma^2)\). Su densidad es: \[ f_X(x) = \frac{1}{x\sqrt{2\pi}\sigma} \exp\left\{-\frac{1}{2}\left(\frac{\log x - \mu}{\sigma}\right)^2\right\}, \quad x > 0 \]

Explora la distribución Lognormal

0.0 0.50
Propiedades de la Lognormal:
  • \(\mathbb{E}[X] = \exp\left\{\mu + \frac{\sigma^2}{2}\right\}\)
  • \(\operatorname{var}(X) = (e^{\sigma^2} - 1) e^{2\mu + \sigma^2}\)
  • \(\mathbb{E}[X^r] = \exp\left\{r\mu + \frac{1}{2}r^2\sigma^2\right\}\) (todos los momentos finitos)
  • No tiene FGM (crece demasiado rápido)
Demostración de los momentos:
Como \(\log X = Y \sim N(\mu, \sigma^2)\), la FGM de \(Y\) es \(m_Y(t) = e^{\mu t + \frac{1}{2}\sigma^2 t^2}\).
Entonces \(\mathbb{E}[X^r] = \mathbb{E}[e^{rY}] = m_Y(r) = \exp\left\{r\mu + \frac{1}{2}r^2\sigma^2\right\}\).
¿Por qué la lognormal no tiene FGM si tiene todos los momentos finitos?
La FGM es \(m_X(t) = \mathbb{E}[e^{tX}]\). Para la lognormal, \(e^{tX}\) crece como \(e^{t e^Y}\), que es super-exponencial. La integral diverge para cualquier \(t > 0\). Este es un contraejemplo clásico: tener todos los momentos finitos no garantiza la existencia de la FGM.

3. Transformación Integral de Probabilidad

Teorema fundamental de simulación:
Si \(X\) es una variable continua con función de distribución \(F_X\) estrictamente creciente, entonces \(U = F_X(X) \sim \text{Unif}(0,1)\).
Recíprocamente, si \(U \sim \text{Unif}(0,1)\), entonces \(X = F_X^{-1}(U)\) tiene distribución \(F_X\).

Demostración interactiva: de uniforme a cualquier distribución

Demostración del teorema:
Para \(0 < u < 1\): \[ F_U(u) = \mathbb{P}(U \le u) = \mathbb{P}(F_X(X) \le u) = \mathbb{P}(X \le F_X^{-1}(u)) = F_X(F_X^{-1}(u)) = u \] Por lo tanto, \(U \sim \text{Unif}(0,1)\).
Recíprocamente, si \(U \sim \text{Unif}(0,1)\) y \(X = F_X^{-1}(U)\), entonces: \[ F_X(x) = \mathbb{P}(X \le x) = \mathbb{P}(F_X^{-1}(U) \le x) = \mathbb{P}(U \le F_X(x)) = F_X(x) \]
Ejemplo: Para generar una exponencial con media θ, tomamos \(U \sim \text{Unif}(0,1)\) y hacemos \[ X = F_X^{-1}(U) = -\theta \log(1-U) \] Este es el método empleado en el simulador del Módulo 1.
¿Por qué es importante este teorema para la simulación computacional?
Es la base de casi todos los generadores de números aleatorios no uniformes. La mayoría de los lenguajes de programación generan primero uniformes (fáciles de generar) y luego los transforman mediante \(F_X^{-1}\) para obtener muestras de cualquier distribución.

4. Transformaciones Multivariadas y el Jacobiano

Para transformaciones de vectores, la fórmula incluye el determinante jacobiano: \[ f_Y(y) = f_X(h(y)) \cdot |J_h(y)| \] donde \(J_h(y) = \det\left( \frac{\partial h_i}{\partial y_j} \right)\).

Visualización: Transformación lineal de normales independientes

Ejemplo del PDF: \(X_1 = Z_1 + Z_2\), \(X_2 = Z_1 - Z_2\) con \(Z_i \sim N(0,1)\) i.i.d.
La transformación inversa: \(Z_1 = \frac{X_1+X_2}{2}\), \(Z_2 = \frac{X_1-X_2}{2}\)
Jacobiano: \[ J = \det\begin{pmatrix} 1/2 & 1/2 \\ 1/2 & -1/2 \end{pmatrix} = -\frac{1}{2} \] \[ f_X(x) = f_Z(h(x)) \cdot |J| = \frac{1}{2\pi} e^{-\frac{1}{2}\left[\left(\frac{x_1+x_2}{2}\right)^2 + \left(\frac{x_1-x_2}{2}\right)^2\right]} \cdot \frac{1}{2} \] Simplificando: \(f_X(x) = \frac{1}{\sqrt{2\pi}\sqrt{2}} e^{-x_1^2/4} \cdot \frac{1}{\sqrt{2\pi}\sqrt{2}} e^{-x_2^2/4}\)
¡\(X_1\) y \(X_2\) son independientes \(N(0,2)\)!
Ejemplo del PDF: Transformación de uniformes a un rombo
Sean \(U_1, U_2 \sim \text{Unif}(0,1)\) independientes. Definimos \(X_1 = U_1 + U_2\), \(X_2 = U_1 - U_2\).
El soporte resultante es un rombo (ver figura en el PDF). La densidad conjunta es uniforme sobre el rombo: \[ f_X(x) = \frac{1}{2} \mathbb{1}_{\{ |x_1| + |x_2| \le 2, |x_1| \le 2, |x_2| \le 1 \}}(x) \]

5. Distribución t de Student

La distribución t de Student con \(n\) grados de libertad se define como: \[ T = \frac{Z}{\sqrt{Y/n}}, \quad \text{donde } Z \sim N(0,1),\; Y \sim \chi^2_n \text{ independientes} \] Su densidad es: \[ f_T(t) = \frac{\Gamma\left(\frac{n+1}{2}\right)}{\sqrt{n\pi}\,\Gamma\left(\frac{n}{2}\right)} \left(1 + \frac{t^2}{n}\right)^{-\frac{n+1}{2}}, \quad t \in \mathbb{R} \]

Comparativa: t-Student vs Normal

5
Propiedades de la distribución t:
  • Simétrica alrededor de 0 (colas más pesadas que la normal)
  • \(\mathbb{E}[T] = 0\) para \(n > 1\)
  • \(\operatorname{var}(T) = \frac{n}{n-2}\) para \(n > 2\)
  • Caso particular: \(t_1 \sim \text{Cauchy}(0,1)\) (varianza infinita)
  • Cuando \(n \to \infty\), \(t_n \xrightarrow{D} N(0,1)\)
Derivación de la densidad t (resumida):
Usando la transformación \((T, U) = \left( \frac{Z}{\sqrt{Y/n}}, Y \right)\) con Jacobiano \(|J| = \sqrt{u/n}\).
Integrando sobre \(u\) se obtiene la forma cerrada de la densidad.
¿Por qué la t tiene colas más pesadas que la normal?
Porque \(Y\) (la chi-cuadrada en el denominador) introduce variabilidad adicional. Para grados de libertad pequeños, hay una probabilidad no despreciable de obtener valores pequeños de \(Y\), lo que hace que \(T\) sea más grande en magnitud.

6. Transformación Box-Muller

La transformación Box-Muller (1958) permite generar dos normales estándar independientes a partir de dos uniformes independientes: \[ X_1 = \sqrt{-2\log U_1} \cos(2\pi U_2), \quad X_2 = \sqrt{-2\log U_1} \sin(2\pi U_2) \]

Generador Box-Muller interactivo

Demostración (esquema):
1. Primera transformación: \(R = \sqrt{-2\log U_1}\), \(\Theta = 2\pi U_2\)
Esto da \((R, \Theta)\) con densidad \(f_{R,\Theta}(r,\theta) = \frac{r}{2\pi} e^{-r^2/2}\) para \(r>0, 0<\theta<2\pi\).
2. Cambio a coordenadas cartesianas: \(X_1 = R\cos\Theta\), \(X_2 = R\sin\Theta\)
El Jacobiano de \((R,\Theta) \to (X_1,X_2)\) es \(1/r\), dando \(f_{X_1,X_2}(x_1,x_2) = \frac{1}{2\pi} e^{-(x_1^2+x_2^2)/2}\).
3. Esta es la densidad conjunta de dos normales estándar independientes.
Importancia histórica: Antes de algoritmos más eficientes, Box-Muller era el método estándar para generar normales. Hoy se usan métodos como el polar de Marsaglia o Ziggurat, pero Box-Muller sigue siendo elegante y didáctico.
¿Qué forma tiene la distribución conjunta de (X₁, X₂) generada por Box-Muller?
Es una normal bivariada con media (0,0), varianzas 1 y correlación 0. Es decir, \(X_1\) y \(X_2\) son independientes \(N(0,1)\). ¡El gráfico de dispersión debe mostrar una nube circular!
